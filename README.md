# Messenger App

Мессенджер на Android с архитектурой Clean Architecture и бэкендом на FastAPI + PostgreSQL.


## О проекте

Учебный проект, разрабатываемый с целью изучения клиент-серверной архитектуры, работы с базами данных и построения масштабируемых мобильных приложений.

На данный момент реализована базовая функциональность мессенджера, продолжается работа над дополнительными возможностями.


## Реализованная функциональность

- Регистрация и аутентификация пользователей
- Личные чаты между пользователями
- Список контактов
- Offline-режим с кэшированием данных (Room)
